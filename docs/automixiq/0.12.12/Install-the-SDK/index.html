    Install the SDK¶

    Requirements¶

    This SDK requires a minimum of Android API Level 21.

    Importing artifacts¶

    AutomixIQ can be brought into your project via our private maven repository.

    Step 1: Include Tuned Global's maven repository

    settings.gradle.kts
    dependencyResolutionManagement {
     repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
     repositories {
         // ...
         maven(url = "https://tunedglobal.pkgs.visualstudio.com/TunedAutomix.Android.Releases/_packaging/AutoMixAndroidFeed/maven/v1")
         // ...
     }
}

    Step 2: Add the dependency to your target

    build.gradle.kts
    // Please check Tuned Global's maven repository for the latest available version
val automixIQVersion = myAutomixIQVersion // 0.5.0

dependencies {
    // ...
    implementation("com.tunedglobal:automixiq:$automixIQVersion")
    // ...
}

    Step 3: You should now be able to use AutomixIQ

    import com.tunedglobal.automix.*

    Configure Proguard (Optional)¶

    If you have any issues finding classes in the SDK, try adding the following to your Proguard configuration.

    -keep class com.tunedglobal.automix.logic.cpp.AMJNILib {
    public static *;
    public *;
}
