    Update UI¶

    AutomixIQ provides realtime status update and playback events after the player starts playing. You can register multiple listeners to monitor the status of the player and update your UI accordingly.

    Register status listener¶

    You can register a status listener any time after the SDK is initialised.

    AutomixSDK.getInstance()?.addStatusListener(myStatusListener)

    Unregister status listener¶

    You can unregister the listener and stop receiving status updates and events when you do not need it any more.

    AutomixSDK.getInstance()?.removeStatusListener(myStatusListener)

    Implement status listener¶

    Extend the AutomixSDKStatusListener class and implement the required functions as shown below:

    • onStatusChanged
    • onPlaybackEvent
    • onWaveformDataAvailable
    class MyStatusListener: AutomixSDKStatusListener {

    override fun onStatusChanged(variables: List<AutomixSDKStatusVariable>, status: AutomixSDKStatus) {
        // Monitor status update
    }

    override fun onPlaybackEvent(type: AutomixSDKPlaybackEventType,
                                 identifier: String?, 
                                 originalSessionId: String?, 
                                 status: AutomixSDKStatus?) {
        // Monitor playback event
    }

    override fun onWaveformDataAvailable(identifier: String) {
        // Monitor waveform data availability
    }
}

    Monitor status update¶

    AutomixIQ reports the player status every time it is updated. You can get the current player status from the AutomixSDKStatus object and the list of AutomixSDKStatusVariable that gets updated since the last time a onStatusChanged is invoked. Use the value of each AutomixSDKStatusVariable to update your player UI.

    For details of each status field, see AutomixSDKStatus and AutomixSDKStatusVariable

    var latestStatus: AutomixSDKStatus? = null

override fun onStatusChanged(variables: List<AutomixSDKStatusVariable>, status: AutomixSDKStatus) {
    // Monitor status update

    this.latestStatus = status

    for (property in variables) {    
        when (property) {
            AutomixSDKStatusVariable.PLAYING -> {
                // Change the look and feel of your Play/Pause button here
                val playing = status.playing
                // ...
            }
            AutomixSDKStatusVariable.MIXING -> {}
            AutomixSDKStatusVariable.PLAYING_TRACK_IDENTIFIER -> {}
            AutomixSDKStatusVariable.PLAYING_TRACK_INDEX -> {}
            AutomixSDKStatusVariable.PLAYING_TRACK_CURRENT_TIME -> {
                // Change your track progress UI here 
                val elapsedTime = status.playingTrackCurrentTime
                val totalTime = status.playingTrackTotalTime
                // ...
            }
            AutomixSDKStatusVariable.PLAYING_TRACK_STREAMING_PROGRESS -> {}
            AutomixSDKStatusVariable.PLAYING_TRACK_TOTAL_TIME -> {
                // Change your track progress UI here 
                val elapsedTime = status.playingTrackCurrentTime
                val totalTime = status.playingTrackTotalTime
                // ...
            }
            AutomixSDKStatusVariable.CAN_SKIP_TO_PREVIOUS -> {
                // Change the look and feel of your previous button here
                val canSkipToPrevious = status.canSkipToPrevious
                // ...
            }
            AutomixSDKStatusVariable.CAN_SKIP_TO_NEXT -> {
                // Change the look and feel of your next button here
                val canSkipToNext = status.canSkipToNext
                // ...
            }
            AutomixSDKStatusVariable.CAN_PLAY_PAUSE -> {
                // Change the look and feel of your play/pause button here
                val canPlayPause = status.canPlayPause
                // ...
            }
            AutomixSDKStatusVariable.CAN_SEEK -> {}
            AutomixSDKStatusVariable.CAN_REORDER_TRACKS -> {}
            AutomixSDKStatusVariable.REPLAY_ON -> {
                // Change the look and feel of your replay button here
                val replayOn = status.replayOn
                // ...
            }

            // ...
            // More `AutomixSDKStatusVariable` enums can be found in API reference
        }
    }
}

    A helper function have been created to help you easily get the value of specific AutomixSDKStatusVariable from AutomixSDKStatus

    Get value for AutomixSDKStatusVariable
    AutomixSDKStatus.getValue<T>(AutomixSDKStatusVariable)

    Monitor playback event¶

    Certain predefined playback events will be returned during playback to help with reporting.

    For details of each playback event, see AutomixSDKPlaybackEventType

    override fun onPlaybackEvent(type: AutomixSDKPlaybackEventType,
                             identifier: String?, 
                             originalSessionId: String?, 
                             status: AutomixSDKStatus?) {
    // Monitor playback event
    when (type) {
        AutomixSDKPlaybackEventType.TRACK_PLAY_START -> {}
        AutomixSDKPlaybackEventType.TRACK_PLAY_PLAYED_5_SECS -> {}
        AutomixSDKPlaybackEventType.TRACK_PLAY_HEARTBEAT_30_SECS -> {}
        AutomixSDKPlaybackEventType.TRACK_PLAY_START_MIXING_INTO_NEXT -> {}
        AutomixSDKPlaybackEventType.TRACK_PLAY_SKIPPED -> {}
        AutomixSDKPlaybackEventType.TRACK_PLAY_FINISHED -> {}
        AutomixSDKPlaybackEventType.MIX_PLAYBACK_ENDED -> {}
    }
}
