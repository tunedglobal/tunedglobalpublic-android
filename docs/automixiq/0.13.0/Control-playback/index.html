    Control playback¶

    AutomixIQ's player needs to be turned on before using any of the following playback controls API, see Turn AutomixIQ on and off.

    Use the AutomixSDK instance to control playback.

    AutomixSDK.getInstance()

    Playback control APIs include:

    • Load tracks
    • Add track

    • Move track
    • Reorder tracks (Shuffle)
    • Clear playlist

    Load tracks¶

    Loads a new set of tracks into the AutomixIQ player, it will override the currently loaded playlist.

    Note

    Please make sure you adhere to the Audio file requirements before loading any tracks to AutomixIQ

    val identifiers: List<String> = myPlaylist 
val metadataList: List<AutomixSDKTrackModel> = mySetOfMetadata
val index = myPlaylistIndex
val startPositionSeconds = myTrackStartPosition
val contextType: String = myContextType
val contextId: String = myContextId

AutomixSDK.getInstance()?.loadTracks(
    identifiers = identifiers,
    metadata = metadataList,
    playTrackIndex = index.coerceAtLeast(0),
    seconds = startPositionSeconds,
    autoplay = true,
    contextType = contextType,      // for reporting purposes
    contextId = contextId,          // for reporting purposes
    onComplete = { automixSDKStatus ->
        // Success
    },
    onError = { automixSDKError, automixSDKStatus ->
        // Error
    }
)

    Add track¶

    Add a track to the current playlist at a given index

    val trackId: String = myTrackId
val trackMetadata: AutomixSDKTrackModel = myTrackMetadata
val index = myIndexToAddTrack

AutomixSDK.getInstance()?.addTrack(trackId, trackMetadata, index, 
    onComplete = { automixSDKStatus ->
        // Success
    },
    onError = { automixSDKError, automixSDKStatus ->
        // Error
    }
)

    Clear playlist¶

    Clears the playlist loaded in AutomixIQ's player, which also stops music playback.

    AutomixSDK.getInstance()?.resetPlaylistToEmpty(
    onComplete = { automixSDKStatus -> 
        // Success
    }, onError = automixSDKError, automixSDKStatus -> {
        // Error
    }
)

    Force pause after playback of current track¶

    Force the player to pause after the current playing track finishes. This will prevent AutomixIQ mixing into the next track.

    This is useful when you would like to play an Ad or do something else in between tracks.

    AutomixSDK.getInstance()?.playTrackUntilEndThenPause(
    onComplete = { automixSDKStatus ->
        // Success
    },
    onError = { automixSDKError, automixSDKStatus ->
        // Error
    }
)
