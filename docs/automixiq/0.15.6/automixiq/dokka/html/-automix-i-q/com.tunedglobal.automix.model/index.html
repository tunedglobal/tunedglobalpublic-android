AutomixIQPluginMetadataModel

class AutomixIQPluginMetadataModel(type: String?, meta: String?)

AutomixSDKPluginMetadataModel models metadata of a track traditionally obtained by running plugins on it. Key and BPM are recommended for best mixing. The Type entry for bpm and key are "key" and "bpm" (case sensitive) respectively, the values need to be formatted as strings, and the key needs to be in OpenKey format.
