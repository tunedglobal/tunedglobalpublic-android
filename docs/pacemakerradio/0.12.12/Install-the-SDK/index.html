    Install the SDK¶

    Requirements¶

    This SDK requires a minimum of Android API Level 21.

    Importing artifacts¶

    PacemakerRadio can be brought into your project via our private maven repository.

    Step 1: Include Tuned Global's maven repository

    settings.gradle.kts
    dependencyResolutionManagement {
     repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
     repositories {
         // ...
         maven(url = "https://tunedglobal.pkgs.visualstudio.com/TunedAutomix.Android.Releases/_packaging/AutoMixAndroidFeed/maven/v1")
         // ...
     }
}

    Step 2: Add the dependency to your target

    build.gradle.kts
    // Please check Tuned Global's maven repository for the latest available version
val pacemakerRadioVersion = myPacemakerRadioVersion // 0.10.4

dependencies {
    // ...
    implementation("org.jetbrains.kotlin:kotlin-reflect")
    implementation("org.greenrobot:eventbus:3.3.1")
    implementation("com.tunedglobal:radiosdk:$pacemakerRadioVersion")
    // ...
}

    Step 3: You should now be able to use PacemakerRadio

    import com.tunedglobal.radio.*
