    Update UI¶

    SocialRadio provides realtime status update and playback events after the player starts playing. You can register multiple listeners to monitor the status of the player and update your UI accordingly.

    Register status listener¶

    You can register a status listener any time after the SDK is initialised.

    SocialRadio.getInstance()?.addStatusListener(myStatusListener)

    Unregister status listener¶

    You can unregister the listener and stop receiving status updates and events when you do not need it any more.

    SocialRadio.getInstance()?.removeStatusListener(myStatusListener)

    Implement status listener¶

    Extend the SocialRadioStatusListener class and implement the required functions as shown below:

    • onStatusChanged
    class MyStatusListener: SocialRadioStatusListener {

    override fun onStatusChanged(variables: List<AutomixIQStatusVariable>, status: AutomixIQStatus) {
        // Monitor status update
    }
}

    Monitor status update¶

    SocialRadio reports the player status every time it is updated. You can get the current player status from the SocialRadioStatus object and the list of SocialRadioStatusVariable that gets updated since the last time a onStatusChanged is invoked. Use the value of each AutomixIQStatusVariable to update your player UI.

    For details of each status field, see SocialRadioStatus and SocialRadioStatusVariable

    var latestStatus: SocialRadioStatus? = null

override fun onStatusChanged(variables: List<SocialRadioStatusVariable>, status: SocialRadioStatus) {
    // Monitor status update

    this.latestStatus = status

    for (property in variables) {    
        when (property) {
            SocialRadioStatusVariable.PLAYING -> {}
            SocialRadioStatusVariable.MIXING -> {}
            SocialRadioStatusVariable.PLAYING_TRACK_IDENTIFIER -> {}
            SocialRadioStatusVariable.PLAYING_TRACK_INDEX -> {}
            SocialRadioStatusVariable.PLAYING_TRACK_CURRENT_TIME -> {}
            SocialRadioStatusVariable.PLAYING_TRACK_TOTAL_TIME -> {}
            SocialRadioStatusVariable.PLAYLIST -> {}
            SocialRadioStatusVariable.FAILED_TRACK_LOAD_IDENTIFIERS -> {}
            SocialRadioStatusVariable.EXPLICIT_LYRICS_FILTERING -> {}
            SocialRadioStatusVariable.MIX_ID -> {}
            SocialRadioStatusVariable.PLAYBACK_MODE -> {}
        }
    }
}

    A helper function have been created to help you easily get the value of specific SocialRadioStatusVariable from SocialRadioStatus

    Get value for AutomixIQStatusVariable
    SocialRadioStatus.getValue<T>(AutomixIQStatusVariable)
